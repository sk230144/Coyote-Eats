# **App Name**: Coyote Eats

## Core Features:

- Interactive Menu: Display the full restaurant menu with high-quality images, descriptions, and prices.
- Online Ordering: Enable customers to place orders online for dine-in, takeaway, or delivery.
- Reservation System: Allow customers to book tables online with a real-time availability checker.
- Photo Gallery: Showcase high-quality photos of the restaurant's ambiance, dishes, and events.
- Customer Reviews: Display customer reviews and ratings, allowing users to submit their own feedback.
- AI Menu Recommendation: AI Tool to provide personalized food and drink recommendations based on customer preferences using a provided LLM tool.
- Restaurant Information: Show the restaurant's address, phone number, opening hours, and a link to the menu. Also show accessibility and LGBTQ+ friendliness.

## Style Guidelines:

- Primary color: Warm beige (#F5F5DC) to evoke a cozy and welcoming atmosphere, inspired by the restaurant's location and name.
- Background color: Soft tan (#E3DBCB), a lighter, desaturated variant of the primary.
- Accent color: Muted gold (#B8860B) to add a touch of warmth and sophistication.
- Body font: 'Alegreya', a serif font for a warm, friendly, literary style; to be paired with Belleza for the headlines.
- Headline font: 'Belleza' a humanist san-serif.
- Use clean, minimalist icons for navigation and key features. Icons should be related to food, drinks, and the restaurant's location.
- A clean, user-friendly layout with a focus on visual content. The menu and photo gallery should be prominently displayed.
- Subtle animations and transitions to create a smooth and engaging user experience.
